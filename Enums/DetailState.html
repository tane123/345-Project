DetailState

enum DetailState

DetailState is an enum that is used to control the state of the DetailReceiptView, relating to each of the buttons present in the view.

  • none

    none: When this is active it will present the view in its default view, with nothing active.

    image: When this is active it will present the image of the receipt.
